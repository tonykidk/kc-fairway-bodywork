How to set up and configure a dev environment

## Github account
Create a github account: [github.com/signup](github.com/signup)

## NodeJS/npm
Install NVM for windows: [https://github.com/coreybutler/nvm-windows](https://github.com/coreybutler/nvm-windows)

Install node:
```
nvm install latest
nvm use latest
```

Verify installation:
```
node -v
```

## Git and Sourcetree for Git UI
Install sourcetree: [https://www.sourcetreeapp.com/](https://www.sourcetreeapp.com/) (will need to create an atlassian account to sign in)

## Cursor editor
Install Cursor: [https://cursor.com/download](https://cursor.com/download)

## Configure github repo
Check out the github repository with sourcetree: [https://github.com/slimnate/sincerely-selfcare](https://github.com/slimnate/sincerely-selfcare)
